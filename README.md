# FlagColoring
A python adaptation of the Combinatorial Game "Flag Coloring"

To play this version, you click on the section you wish you change and use keys to pick the color you wish to change it to.  
The keys are:
*  'r' = Red
*  'g' = Green
*  'b' = Blue
*  'y' = Yellow
*  'w' = White
*  'k' = Black

# Requirements  
* pygame
